# Cross-Post Extension for X & Bluesky

A Chrome extension that allows you to easily cross-post between X (formerly Twitter) and Bluesky. Simply check a box next to the post button to automatically copy your post to the other platform.

## Installation

### From Source (Developer Mode)
1. Clone this repository or download the ZIP file
   ```bash
   git clone https://github.com/fberrez/blueskix
   ```

2. Open your Chromium browser and go to:
   - Chrome: `chrome://extensions/`
   - Brave: `brave://extensions/`
   - Edge: `edge://extensions/`

3. Enable "Developer mode" in the top-right corner

4. Click "Load unpacked" and select the extension directory

## Usage

1. Visit X (x.com) or Bluesky (bsky.app)
2. Write your post as usual
3. Check the cross-post checkbox that appears next to the post button
4. Click the post button
5. A new tab will open with your post text ready to be posted on the other platform

## Known Limitations

- Due to X's API restrictions, this extension uses a manual approach
- Media (images, videos) cross-posting is not supported yet
- The extension needs both sites to be accessible in your browser
